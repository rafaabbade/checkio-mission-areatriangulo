Enunciado

Lúcia, uma engenheira civil, está projetando diversas áreas de lazer em diferentes cidades. Ela precisa de uma ferramenta que calcule rapidamente a área de triângulos para ajudar no planejamento do uso do espaço. A ferramenta deve ser flexível o suficiente para aceitar quaisquer dimensões fornecidas.

Entrada: Dois números inteiros: a base e a altura do triângulo.

Saída: Um número float indicando a área do triângulo.
